  - [ ] Сервис загрузки аудиодорожки.
- [ ] **3.2. AI-интеграция (Service Layer):**
  - [ ] Интеграция Yandex SpeechKit (или Whisper) для ASR (Audio-to-Text).

    - [ ] Тестов (Quiz) с таймкодами.
- [ ] **3.3. Видеоплеер (Frontend Feature):**
  - [ ] Кастомный плеер (на базе `react-player` или HTML5 Video).
